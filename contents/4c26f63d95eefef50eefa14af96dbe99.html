Login

UMLUseCase
Projeto 1 :: Model :: Login

Description

+Fluxo Principal

  1. Usuário entra com login.[F001]
  2. Usuário entra com a senha.[FS002]
  3. Entrada autorizada.
  4. A data e hora do login é atualizada.

+Fluxo Secundário [FS001]

  1. O usuário entra com seu login.
  2. O usuário digital seu e-mail para recuperar a senha.

+Fluxo Secundário [FS002]

  1. Senha inserida pelo usuário é incorreta.
  2. Usuário inseriu a senha incorreta várias vezes.
  3. Usuário tem o acesso bloqueado.
  4. Um alerta via e-mail ou SMS será enviado para informar o usuário a situação de sua conta.
