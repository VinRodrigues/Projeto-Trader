Ordem de Venda

UMLUseCase
Projeto 1 - Diagrama de caso de uso :: Model :: Ordem de Venda

Description

+Fluxo Principal

  1. Usuário acessa a interface de ativos[FS001]
  2. Usuário verifica valores de ativos.
  3. O usuário seleciona o ativo que deseja vender.
  4. O usuário visualiza um gráfico infraday e histórico da ação, o gráfico mostra o valor atual, abertura, fechamento anterior, máximo e mínimo.
  5. O usuário envia ordens de venda.[FS002]
  6. B3 avalia ordem de venda.
  7. B3 autoriza a ordem de venda.

+Fluxo Secundário[FS001]

  1. Usuário visualiza ativos comprados.
  2. Usuário checa o status da OV.

+Fluxo Secundário[FS002]

  1. Usuário solicita OV por quantidade de ações ou por valor financeiro.
  2. Usuário checa o status da OV.
  3. Usuário visualiza OV relacionada a ação relacionada.
