Modificação de OV

UMLUseCase
Projeto 1 - Diagrama de caso de uso :: Model :: Modificação de OV

Description

+Fluxo Principal.

  1. O usuário entra a interface de interação de ativos.
  2. O usuário acessa área de seus ativos comprados.[FS001]
  3. O usuário seleciona um de seus ativos.[FS002]
  4. O usuário seleciona a modificação da OV.

+Fluxo Secundário[FS001]

1.Usuário entra em seu perfil. 2. Usuário acessa o seu histórico de OV.

+Fluxo Secundário [FS002]

  1. O usuário seleciona a opção de cancelamento de OV de ativos.
  2. O usuário aguarda a autorização do S2.
  3. O S2 autoriza o cancelamento de ordem de compra.
  4. Usuário confirma o cancelamento da OV.
