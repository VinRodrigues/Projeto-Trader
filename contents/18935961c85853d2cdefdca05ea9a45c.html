Visualizar Carteira de ações

UMLUseCase
Projeto 1 :: Model :: Visualizar Carteira de ações

Description

+Fluxo Principal

  1. O usuário acessa a carteira de ações.[F001]
  2. O usuário visualiza OC e OV, como opções em sua carteira.
  3. O Usuário seleciona a opção OC.[F002]
  4. O usuário ativa um aviso caso ele esteja negociando o mercado fracionário.
  5. O usuário autoriza compra de ação fracionado/ lote automaticamente.[F003]
  6. O usuário compra lote.
  7. O usuário seleciona a quantidade do lote
  8. O usuário recebe um aviso de compra no mercado fracionário.

+Fluxo Secundário.[F001]

  1. O usuário seleciona opção de extrato em sua carteira.
  2. O usuário recebe o extrato das ações com todos os negócios realizados com ela em um determinado período de tempo.

+Fluxo Secundário.[F002]

  1. O Usuário seleciona a opção OV.
  2. O usuário ativa um aviso caso ele esteja negociando o mercado fracionário.
  3. O usuário autoriza venda de ação fracionado/ lote automaticamente
  4. O usuário recebe um aviso de venda no mercado fracionário.

+Fluxo Secundário.[F003] 1. O usuário inicia ordem de compra. 2. A compra não é autorizada, por falta de fundos da carteira. 3. A ordem de compra se encerra.
